Welcome to Rob's Memory Card Game!

Memory Game

Memory Card Game is played by selecting two cards from the 18 cards available and getting matches. The player will select the cards using their mouse and clicking the cards, they will then flip showing what the card image is that needs to be matched. The object of the game is find all of the matches available from the 18 cards. You win if you get all of the cards to match. They will stay flipped with a green background, you lose if you cant get all the matches. Click "replay game" to start over.
